# MySQL

## CRUD(Create Read Update Delete)

### Create

- **데이터 베이스 생성**
    - ******************************`create database`******************************
- ******************유저 생성******************
    - ********************`**create user**`********************
- **************************************테이블 생성**************************************
    - **************************************************************`**create table 테이블명();**`**************************************************************

### Read

- **************************************테이블 컬럼 보기**************************************
    - **********************************`**desc 테이블명**`**********************************
- **************************************테이블 내용 보기**************************************
    - **************************************모든 내용 보기**************************************
        - **********`select * from 테이블명`**********
    - ******************************************내용 선택해서 보기******************************************
        - ****`select 컬럼명 from 테이블명`****
    - ************************************특정 조건 걸기************************************
        - ********`where 특정 컬럼 = 값`********

### Update

- ****************************************테이블 내용 변경****************************************
    - ************************************`update 테이블명 set 바꿀 부분 = 바꿀 내용`************************************
- ************************************특정 조건 걸기************************************
    - **********`where 특정 컬럼 = 값`**********

### Delete

- ************************테이블 삭제************************
    - **********************************`**drop 테이블명**`**********************************
- **********************************************************************************특정 테이블에서 특정 데이터 삭제**********************************************************************************
    - ********************************************************************************************************`**delete from 테이블명 where 특정 컬럼 = 값**`********************************************************************************************************
